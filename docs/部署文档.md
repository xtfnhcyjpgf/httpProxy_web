## 部署架构

### 后端服务
- 使用 Gunicorn 作为 WSGI 服务器
- 监听端口: 8090
- Systemd 服务管理: httpProxy_web.service

### 前端服务
- 使用 Nginx 提供静态文件服务
- 监听端口: 8080
- 同时作为 API 反向代理

### 文件结构（服务器）
```
/var/www/httpProxy_web/
├── app.py
├── config.py
├── database.py
├── requirements.txt
├── venv/              # Python 虚拟环境
├── frontend/
│   └── dist/
└── uploads/
```

## 手动部署步骤（备用）

### 1. 上传项目文件到服务器
```bash
scp -r * root@8.137.176.50:/var/www/httpProxy_web/
```

### 2. 安装系统依赖
```bash
ssh root@8.137.176.50
apt update
apt install -y python3-pip python3-venv nginx
```

### 3. 设置 Python 环境
```bash
cd /var/www/httpProxy_web
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt gunicorn
```

### 4. 配置 Systemd 服务
创建文件: `/etc/systemd/system/httpProxy_web.service`
```ini
[Unit]
Description=Gunicorn instance to serve httpProxy_web
After=network.target

[Service]
User=root
Group=root
WorkingDirectory=/var/www/httpProxy_web
Environment="PATH=/var/www/httpProxy_web/venv/bin"
ExecStart=/var/www/httpProxy_web/venv/bin/gunicorn --workers 3 --bind 0.0.0.0:8090 app:app

[Install]
WantedBy=multi-user.target
```

### 5. 配置 Nginx
创建文件: `/etc/nginx/sites-available/httpProxy_web`
```nginx
server {
    listen 8080;
    server_name _;

    location / {
        root /var/www/httpProxy_web/frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:8090;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

启用配置:
```bash
ln -s /etc/nginx/sites-available/httpProxy_web /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
```

### 6. 启动服务
```bash
systemctl daemon-reload
systemctl enable httpProxy_web
systemctl start httpProxy_web
systemctl restart nginx
```
